## Descrição:

<p> A aplicação imita funcionalidades do Twitter, permitindo que o usuário se registre, faça login, edite suas informações e poste em sua timeline.</p>

## Tenologias:
- NodeJS com Express
- React JS
- Redux
- MongoDB
